Netflix Clone
I built this project from Youtube using React JS and it was a very educative experience. I made use of TMDB API key.To handle the API calls, I used axios and used the inbuilt React hooks for fetching the data.I also used CSS flexbox amongst other effects to style the UI. For the trailer popups, I made use of the react-Youtube library.

TikTok Clone
This is another project I handled with React. I made use of Material UI whilst using Firebase for the hosting and database. CSS Flexbox also was employed here and because of their importance, the useState and useEffect hooks were also used. I was able to pull the videos in realtime though due to security measures, they do not persist for long.

Zoom Clone
The Pandemic made Zoom an ubiqutious app. So I decided to give it a try. I built this with Nodejs and employed the services; UUid, SocketIO, Express js and WEBRTC for this project on backend.
